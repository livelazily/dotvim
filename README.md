## Installation
1. Clone this repo

```bash
git clone --recursive https://github.com/livelazily/dotvim.git ~/.vim
```

2. Install [vim-plug](https://github.com/junegunn/vim-plug#installation)

3. Install plugins

```bash
vim +PlugInstall +qall
```

## Create symlinks:
```bash
ln -s ~/.vim/vimrc ~/.vimrc
ln -s ~/.vim/gvimrc ~/.gvimrc
```
